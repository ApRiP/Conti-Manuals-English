2)SHELL NET VIEW DOMAIN NAME /all /domain

3)VIEW THE LIST OF DC shell nltest /dclist:"NameDomain"

4)FIND OUT THE LIST OF SERVERS
LOADING THE PowerView MODULE
RIGHT-CLICK ON THE AGENT Get Info > Get Servers
GOT A LIST OF SERVERS

5)FIND OUT THE LIST OF COMPUTERS
BECAUSE THE POWERVIEW MODULE IS ALREADY LOADED
RIGHT CLICK ON THE AGENT Get Info > Get All Computers
GOT A LIST OF COMPUTERS

GO ABOVE THE VIEW TAB > CREDENTIALS
TAKE ALL HASHES AND LOOK FOR DOMAIN ADMINS
